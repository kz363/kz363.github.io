Technical Blog

March 22, 2014 by Kevin

This is a question that I wasn't sure about, but JavaScript has always piqued my interests because I hear it mentioned on the internet all the time. After some searching on Google, which uses tons of JavaScript, these are some of the reasons why JavaScript has become so popular for web development:


JavaScript...

  • Is easy to learn compared to other languages.
  • Easy to edit because it doesn't need to be compiled or written in expensive software. Any simple text editor will do.
  • Has a lot of libraries, which are collections of pre-written code that allows developers to cut down on development time.
  • Allows web pages to be interactive. Sites like Gmail and Google Maps are built with a lot of JavaScript code.
  • Is supported by every modern browser, allowing developers to reach a huge audience with their code.
  • Can interact with all the elements of a web page (HTML & CSS) to make dynamic web pages as opposed to static ones.
  • Runs on the user's computer so it doesn't put extra stress on the server.

All these reasons makes it easy to see why JavaScript is the popular choice, especially for web development. It allows web pages to interact with the user right in their own browser without the need for any additional software. I predict that JavaScript will continue to be a standard in the future as it is now.
